# 简介
图片的文字，想要复制或者摘抄下来，只能繁琐的打一个一个字。为了解决不能摘抄文字的问题，此程序应用而生。用户上传图片，即可立马获取图片的文字。

这类工具的产品适用于开发在小程序的平台，方便快捷，用完即走。

## 使用操作
在微信小程序搜索**图像文字识别plus**，或扫描下方二维码：
